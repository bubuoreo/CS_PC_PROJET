# CS_PC_PROJET
Auteurs : BURLOT Alexandre et ROTH Lucas
Dates : début le 25 mai 2021 jusqu'au 18 juin 2021
Lien github : https://github.com/bubuoreo/CS_PC_PROJET

Introduction :

Au sein de ce projet, vous pourrez voir les réalisation que les auteurs ont pu réaliser durant les séances de TP allouées ainsi que pendant leur temps libre.
5 sujets ont été traité pour la réalisation de ce projet que sont :
 - La course hippique
 - Le calcul de PI avec la méthode des arctangentes
 - Une méthode de tri de liste : quick sort
 - Le sujet sur le controle de température
 - La simulation d'un restaurant

L'objectif de ces fichiers étaient de transposer ou créer les programmes décrits dans le cahier des charges en utilisant le parallèlisme et en utilisant la bibliothèque multiprocessing de python.

Bonne continuation.
